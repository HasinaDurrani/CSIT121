What I Learned From the Labs

This semester has been a rewarding journey through web standards and best practices. Each lab provided valuable insights into different aspects of web development.

In Lab 1, I learned the basics of HTML and structuring web pages. Understanding elements, attributes, and how to create an accessible layout laid the foundation for my web design skills.

Lab 2 taught me about CSS and how to style pages effectively. I learned the importance of creating responsive designs using media queries, ensuring that my websites are accessible across different devices.

In Lab 3, I explored the concepts of web accessibility and how to design websites that are usable for everyone, including people with disabilities. This made me realize how crucial it is to implement accessibility features for a more inclusive web.

Lab 4 deepened my knowledge of images and formats, helping me understand the impact of choosing the right image format on a website's performance and user experience.

Lab 5 was about linking and navigation. I learned how to create meaningful links and organize content logically for easier navigation, which is crucial for improving user experience.

Lab 6 introduced me to advanced CSS techniques like Flexbox and Grid. I learned how to design layouts with more flexibility and precision, giving me more control over my page structures.

In Lab 7, I experimented with interactive elements, including forms and buttons. I learned how to create dynamic, user-friendly interfaces that make websites more engaging.

Lab 8 emphasized the importance of responsive web design. I gained practical skills in creating layouts that adapt to various screen sizes, which is vital in today's mobile-first world.

Lab 9 involved integrating XML with HTML. This taught me how to dynamically load and display data on a webpage, which is crucial for building more complex, data-driven websites.

Finally, Lab 10 focused on understanding the DOM and how JavaScript interacts with HTML elements. I learned how to manipulate the DOM to create interactive and dynamic web pages.

What the Web Standards Course Taught Me

This course has been an eye-opener into the world of web development. It taught me the importance of adhering to web standards, which ensures consistency, accessibility, and performance across various browsers and devices. I also learned the significance of responsive design, accessibility, and best practices in coding and structuring websites.

Each lab reinforced the importance of planning and designing websites that are both functional and user-friendly. It taught me that web development isn't just about writing code, but about creating meaningful, accessible experiences for users.

Overall, this course has equipped me with the necessary skills to develop well-structured, accessible, and responsive websites that meet modern web standards.
