Website Evaluation - Baltimore County Public Schools

Website URL

https://www.bcps.org/

Website Name

Baltimore County Public Schools (BCPS)

Target Audience

The target audience includes students, parents, staff, and community members in Baltimore County interested in school district information.

Site Organization

The BCPS site is organized with a top navigation menu, providing easy access to sections like Schools, Students, Parents, Staff, and more. It uses a clean layout with clear headings to separate content sections.

CRAP Design Principle

Contrast: The website uses contrasting colors for links and buttons, making them stand out against the white background, like the orange 'Safe Schools Link' button.

Audit Score

The website achieved an audit score of 92 according to the Accessibility Checker, showing a strong commitment to accessibility standards.

Effectiveness

The BCPS website is effective, supporting users in completing actions like accessing the school calendar, finding staff directories, and viewing news updates with ease.

Efficiency

Users can perform tasks quickly on this site, as it provides a straightforward navigation menu and clear labels for each section.

Engagement

The site is pleasant to use, with a user-friendly design that suits its educational and informative purpose, making it engaging for the target audience.

Recommendation

One recommendation to improve the site would be to increase the font size slightly to enhance readability for all users, especially on mobile devices.
