Lab 5 Reflection

Published on October 6, 2024 by Hasina Durrani

In this lab, I learned how to create a responsive design using CSS Flexbox and media queries. It was exciting to see how the layout changed based on the screen size. I also implemented internal navigation, which made the page easier to navigate. I realized how important it is for websites to look good on all devices.

Best design practices I learned include keeping the layout simple, using consistent color schemes, and ensuring text is readable on all devices. This experience will help me create more user-friendly designs in the future. I also learned to think about how users will interact with my page, making it easier for them to find what they need.
